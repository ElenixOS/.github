ElenixOS aims to build an Android-like application ecosystem for smartwatches and embedded wearable devices.

Powered by JerryScript and LVGL, ElenixOS allows applications and watch faces to run inside a unified JavaScript runtime, enabling dynamic application deployment and ecosystem expansion beyond traditional firmware-based smartwatch systems.

By combining portable native infrastructure with script-driven applications, ElenixOS provides a consistent development and user experience across multiple hardware platforms.

This organization maintains the ports, simulators, tools, and ecosystem projects that make this vision possible.
